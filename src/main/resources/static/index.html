주사위가 굴러서 시작일까요? 시작 버튼을 눌러서 주사위가 구른 것일까요?
이게 무슨 말인지 의문이 드시는 분이 많으실 것으로 예상합니다.
왜냐면 저는 인프런 김영한 MVC 강의를 듣고 여러 버전의 주사위를 만들 것이기 때문이니까요.
처음에는 해당 페이지에 들어가자마자 바로 숫자만 나오는 주사위 게임이 시작될 것이고,
그 다음부터는 버튼 없이 이미지가 나오는 주사위 게임이 시작될 것입니다.
그래도 시작 버튼 누르고 굴리는 주사위도 구현 할 것이니 걱정 많이 해주세요.

처음에 매뜨 random 함수 배우면서 했던 주사위 게임을 콘솔 창이 아닌 이미지? 구현으로 하고자
했던 것인데 생각보다 일을 크게 벌인 것 같습니다.
2인 게임도 만들고 싶은데 이건 앞으로 노력해보겠습니다.
